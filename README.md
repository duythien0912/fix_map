FixMap là ứng dụng giúp bạn tìm chỗ sửa xe gần bạn nhất

## Các chức năng
- Ứng dụng sử dụng Flutter với mô hình Bloc
- Tìm chỗ  sửa xe quanh đây
- Đánh giá chỗ sửa xe
- Đăng nhập, Đăng xuất
- Giao diện sáng, tối
- Online, offile
- Ngôn ngữ hỗ trợ
